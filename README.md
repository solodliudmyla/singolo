# SINGOLO
---
`HTML5` `CSS3` `Figma` `BEM naming` `Pixel-perfect design` `Responsive design` `JavaScript ES6+` `Without libraries and frameworks`

- CSS naming with BEM methodology. 
- Semantic HTML5 markup.
